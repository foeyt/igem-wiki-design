# 部分工具的安装与使用

## Git

- 安装地址：https://git-scm.com/install/windows；选择 Git for Windows/x64 Setup。
- 下载完成后直接点击安装即可。
- 下面是 Git 常用命令：[常用 Git 命令清单 - 阮一峰的网络日志](https://www.ruanyifeng.com/blog/2015/12/git-cheat-sheet.html)。

## VSCode

- 推荐几个插件：
  - Auto Rename Tag：自动重命名配对的 HTML 标签。
  - ES7+ React/Redux/React-Native snippets：React 开发必备。
  - GitLens — Git supercharged：可视化 Git 操作。
  - Prettier - Code formatter：代码格式化使代码更加美观。
  - vscode-icons：给文件和文件夹加上图标方便查找。

- 直接在 VSCode 中安装即可。
